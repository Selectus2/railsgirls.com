Setting your computer for Rails Girls

Follow the instructions for your operating system. If you hit into any problems, don’t panic. Inform us on at the event and we can solve it together.

  • Setup for OS X
  • Setup for Windows
  • Setup for Ubuntu

Setup for OS X

First we need to install general tools needed to install other developer software.

Step 1. Lets check the version of the operating system. Click the Apple menu and choose About this Mac (Picture 1).

Picture 1

Step 2. In the window you find the version of your operating system (Picture 2). If your version number starts with 10.6 or 10.7 this guide is for you. If it’s something else, we can setup your machine at the event.

Picture 2

Step 3. Download and install these developer tools for OS X depending of your system version number.

  • Installer for 10.7 (273Mb)
  • Installer for 10.6 (273Mb)

Step 4. After the installer has run, open Terminal.app. Click the Spotlight (magnifying glass icon in the top right corner), type Terminal.app and click the option that appears (Picture 3).


Picture 3

Step 5. Now copy and paste this line the Terminal and press Enter. Enjoy the text flying on the screen, it will take quite some time (approximately 30 minutes). Installer might ask for your administrator password. Grabbing a refreshing drink before starting is encouraged!

bash < <(curl -s https://raw.github.com/gist/1214700/rails-install.sh)

Step 6. You also need a text editor to edit code files. For the workshop we recommend free code editor Komodo Edit. Download the app and install it.

Now if everything went right, you should have a working Ruby on Rails programming setup. Congrats!

Setup for Windows

Download RailsInstaller and run it. Click through the installer using the default options.

You also need a text editor to edit code files. For the workshop we recommend free code editor Komodo Edit. Download the installer and run it.

Now you have a working Ruby on Rails programming setup. Congrats!
