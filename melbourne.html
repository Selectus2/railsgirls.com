Rails Girls Melbourne 14th September 2019

Rails Girls is once again coming to Melbourne, bigger and better than ever! During the two day workshop we'll dive into the magical world of Ruby on Rails.

Tweet

This time we'll be running two seperate free one-day workshops at the same time:

Rails Girls

A great starting point if you're new to coding, web applications and Ruby! You'll learn designing, prototyping and coding with the help from our coaches. You don't need any previous coding experience. You do need your own laptop, curiosity and a sprinkle of imagination!

Registration for Rails Girls open now

Rails Girls Next

You've done Rails Girls - now take the next step! We'll be diving even deeper into programming in Ruby, covering topics like Test Driven Development, Agile Software Development and you'll have the chance to workshop the famous toy robot challenge with our wonderful coaches. To participate in this workshop, you're required to have either done the original Rails Girls day, or have some prior programming experience.
